# 케이스 기반 Socratic 사고 촉진 GPT

고등학생 및 대학 초급 학습자를 위한 사회과학 사고 촉진 웹앱입니다.

이 앱은 학생이 사회 문제에 대해 자신의 입장, 근거, 전제, 반대 관점, 현실 사례를 단계적으로 탐색할 수 있도록 돕는 것을 목표로 합니다. 단순히 정답을 알려주는 챗봇이 아니라, 학생이 스스로 생각을 구조화하고 자신의 사고 변화를 돌아볼 수 있도록 설계되었습니다.

## 개발 배경

이 프로젝트는 사회과학 수업에서 학생들이 단순히 정답을 듣는 것이 아니라, 자신의 생각을 드러내고 근거를 점검하며 다양한 관점에서 사회 문제를 이해하도록 돕기 위해 제작되었습니다.

특히 현실 사례와 Socratic 질문 방식을 결합하여, 학생이 스스로 사고를 확장하고 자신의 관점이 어떻게 변화하는지 돌아볼 수 있도록 설계되었습니다. 



## 프로젝트 목적

사회과학 수업에서는 학생이 현실 사회 문제를 단순한 찬반으로만 이해하기보다, 다양한 이해관계자와 가치 충돌을 함께 고려하는 것이 중요합니다.

이 앱은 다음과 같은 사고 흐름을 지원합니다.

```text
현실 맥락 이해
→ 학생 초기 입장 확인
→ 근거 확인
→ 사고 구조화
→ 반대 관점 또는 예외 상황 탐색
→ 실제 사례 연결
→ 교과 개념 정리
→ 메타인지적 마무리
```

## 주요 기능
- 교사용 수업 정보 사전 입력
- 교과 영역
- 활동 유형
- 논의 주제
- 학습 목표
- 학생 답변 기반 Socratic 대화 진행
- 학생이 근거 없이 주장만 제시할 경우 근거 질문 제공
- 학생 답변을 바탕으로 사고 구조화
- 반대 관점, 예외 상황, 이해관계자 차이 등을 통한 관점 확장
- 실제 사회 사례 및 기사 링크 기반 사고 확장
- 교재 또는 통계자료 이미지 업로드 가능
- 대화 흐름 마무리 및 협동 토론 질문 제안

## 활동 유형

이 앱은 다음 세 가지 활동 유형을 지원합니다.

### 1. 토론형

학생의 입장과 근거를 중심으로 찬성, 반대, 조건부 입장을 탐색합니다.

#### 예시 주제:

- 디지털 장의사를 통한 잊힐 권리는 알 권리보다 우선될 수 있을까?
- 인공 임신 중절에서 여성의 자기결정권은 태아의 생명권보다 우선될 수 있을까?
- AI 감시는 공공 안전을 위해 허용될 수 있을까?

### 2. 개념 이해형

현실 사례 속에서 사회과학 개념을 발견하고 이해하도록 돕습니다.

#### 예시 주제:

- 젠트리피케이션
- 외부효과
- 시장 실패

### 3. 탐구형

사회 현상의 원인, 변수, 이해관계자, 구조적 배경을 탐색합니다.

#### 예시 주제:

- 국제무역은 왜 발생할까?
- 저출산은 왜 해결이 어려울까?
- 지역 격차는 왜 심화될까?

#### 사용 예시

- 교사 입력 예시:
```
교과 영역: 윤리
활동 유형: 토론형
논의 주제: 디지털 장의사를 통한 ‘잊힐 권리’는 알 권리보다 우선될 수 있을까?
학습 목표: 디지털 사회에서 사생활 보호와 알 권리의 충돌을 다양한 관점에서 탐구한다.
```

- 예상 대화 흐름:

```
GPT: 최근에는 디지털 장의사를 통해 과거 게시물이나 검색 결과 삭제를 요청하는 사람들이 늘고 있어.
한편으로는 개인의 사생활을 보호할 수 있지만, 다른 한편으로는 사회적으로 알아야 할 정보까지 사라질 수 있다는 우려도 있어.

디지털 장의사를 통한 ‘잊힐 권리’는 알 권리보다 우선될 수 있다고 생각해?
```

- 학생 답변 예시:
```
나는 조건부 찬성이라고 생각해.
일반인의 과거 실수나 사생활은 보호가 필요하지만, 범죄나 공익과 관련된 정보는 사람들이 알아야 한다고 생각해.
```

이후 GPT는 학생 답변을 바탕으로 근거를 구조화하고, 반대 관점과 현실 사례를 연결합니다.

## 설치 방법

프로젝트를 내려받은 뒤 필요한 패키지를 설치합니다.
```
pip install -r requirements.txt
```

## 실행 방법

터미널에서 아래 명령어를 입력합니다.
```
streamlit run app.py
```

실행 후 브라우저에서 아래 주소로 접속합니다.
```
http://localhost:8501
```
